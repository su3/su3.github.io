设置TableView 背景图

iOS 5 可以这样

myTable.backgroundColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"bgSetting.png"]];

以上方法在 iOS 6 无效，同时支持 iOS 5 & iOS 6 可以这样设置

UIView *tableBg = [[UIView alloc] init];
tableBg.backgroundColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"bgSetting.png"]];
myTable.backgroundView = tableBg;

Section 背景透明

- (UIView *)tableView:(UITableView *)tableView viewForHeaderInSection:(NSInteger)section
{
    UIView *result = [[UIView alloc] init];
    [result setBackgroundColor:[UIColor clearColor]];
    UILabel *label1 = [[UILabel alloc] initWithFrame:CGRectMake(19, 0, 100, 35)];
    label1.backgroundColor = [UIColor clearColor];
    label1.textColor = [UIColor grayColor];
    [result addSubview:label1];
    if (section == 0) {
    label1.text = @"Sounds";
    }else{
    label1.text = @"Screen-Lock";
    }
    return result;
}

TableViewCell 的背景图片

- (void)tableView:(UITableView *)tableView willDisplayCell:(UITableViewCell *)cell forRowAtIndexPath:(NSIndexPath *)indexPath {
cell.backgroundColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"image.png"]];
}

改变分割线的颜色

myTable.separatorColor = [UIColor clearColor];

去掉分割线

myTable.separatorStyle = UITableViewCellSeparatorStyleNone;

改变 UITableViewCellAccessoryCheckmark 的颜色？

UITableViewCellAccessoryCheckmark 的颜色改不了，把 cell.accessoryType 改为设置 cell.accessoryView

UIView *checkedView = [[UIView alloc] initWithFrame:CGRectMake(0, 0, 10, 20)];
checkedView.backgroundColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"checked.png"]];
cell.accessoryView = checkedView;
