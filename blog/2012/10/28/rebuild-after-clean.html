更新资源文件之后 rebuild，仍然输出旧文件？

2012-10-28 09:38
  • Product -> Clean，清除所有编译文件，这一步不是必须的，因为IDE会跟踪所有资源的改变；
  • Product， 按住 option 键，Clean Build Folder ... 清除之前 debug 时生成的 bundle；
  • iOS Simulator 中删除 APP；
  • 如果有必要，在 Build Phases -> Copy Bundle Resources 中把该资源删除，再重新添加。
