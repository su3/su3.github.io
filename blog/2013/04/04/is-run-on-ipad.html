检查 iPhone 版 App 是不是运行在 iPad 上

2013-04-04 22:23

一个APP，做了 iPhone 和 iPad 两个独立版本。我希望当 iPhone 版运行在 iPad 上时，提示用户去下载 iPad版。但是非 universal 版本如果仅判断 UIUSERINTERFACE_IDIOM() ，返回值永远是 UIUserInterfaceIdiomPhone，怎么办呢？这时需要判断 model 类型是否为 iPad

if (UI_USER_INTERFACE_IDIOM() == UIUserInterfaceIdiomPhone &&
[[[UIDevice currentDevice] model] hasPrefix:@"iPad"]) {
// This app is an iPhone app running on an iPad
}
