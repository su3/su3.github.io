获取当前键盘输入法

2013-02-21 21:00

在 viewDidLoad 中加个事件监听

[[NSNotificationCenter defaultCenter] addObserver:self selector:@selector(correntKeyboardLanguage) name:UITextInputCurrentInputModeDidChangeNotification object:nil];

得到通知的方法

- (void)currentKeyboardLanguage{
UITextInputMode *textInput = [UITextInputMode currentInputMode];
NSString *primaryLanguage = textInput.primaryLanguage;
NSLog(@"Current text input is: %@", primaryLanguage);
}

可以得到字符串，比如 en-US 为英文， zh-Hans 为简体中文，zh-Hant 为繁体中文。

参考: how-to-use-uitextinputmode
